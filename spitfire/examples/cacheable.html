Extending cacheable

The class contains to abstract methods:
  • abstract public function generateKey(); returns the key that memcached uses to store this data.
  • abstract public function fallback(); used to get the value if memcached key is not present.
Lets create a example subclass 

	class onlineUsers extends cacheable
	{
		public function generateKey() {
			return 'onlineUsers';
		}
		
		public function fallback() {
			$db = DB();
			return $db->onlineUsers->idUser->count();
		}
	}
Now you can use it like this: 

	$onlineUsers = new onlineUsers();
	$onlineUsers = $onlineUsers->get(): //Returns the actual number
